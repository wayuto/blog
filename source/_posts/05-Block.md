在`Alum`中，`{` `}`的作用不再局限于创建作用域或用于包裹函数体，他可以看作一个有多个表达式组成的复合表达式，这与`Rust`比较类似，同时它与普通的表达式别无二致，并且可以产生值。例如：
```alum
let result: int = {
    let n: int = 1
    let m: int = 2
    n + m
}
```
这时，`result`的只就是`n + m`，由于独立的作用域，`n`与`m`在`{` `}`外不可访问。  
再例如，调用函数时传入的参数也可以是由`{}`包裹的表达式，例如
```alum
F({
    let a: int = 2
    let b: int = 3
    a * b
})
```
运行效果等价于`F(2 * 3)`，但这种语法给了`Alum`很大的灵活性，合理使用可以提升效率，但同时也可能失去一定的可读性。
